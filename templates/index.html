Customer Segmentation

This is an Customer Segmentation Web App that allows you to input csv files consisting datasets to group the customer into clusters based on similar properties

The follows are the simple steps to perform customer segmentation

Step1 : Upload your dataset [CSV or Excel File]

Step2 : Check out for null values,duplicate values and perform imputation.

Step3 : Select Annual Income and Spending Score as an columns.

Step4 : Use that cleaned dataset to proceed to clustering and visulization

Step5 : Clustering will be done by using K-Means Clustering

Step6 : All the results will be visualized on the screen
